# קוויזלטים לבגרות

<div dir="rtl">

אפליקציית כרטיסיות ללימוד **עברית ⇄ ערבית** לקראת **בגרות בערבית**.
PWA נטו (HTML + CSS + JavaScript בודד), פועלת גם **בלי אינטרנט** לאחר טעינה ראשונה, ושומרת את כל הנתונים ב-`localStorage` בדפדפן שלך — בלי שרת, בלי חשבון.

## ✨ מה יש בפנים

- **5 מצבי לימוד:**
  - 🃏 **כרטיסיות** — הפוך כרטיס, סמן "יודע/לא יודע"
  - 📋 **רשימה** — כל החפיסה בבת אחת, עם הסתרת עברית/ערבית
  - 🔤 **השלמה** — השלם מילים חסרות במשפט. הגדרות: כמה חוסרים (1/2/3/הכל), צורת מענה (הקלדה או אמריקאי עם 4 אפשרויות). ההשוואה מתעלמת מניקוד.
  - 🧩 **התאמה** — משחק זיכרון בין מילים בשתי השפות
  - ❓ **אמריקאי** — 4 תשובות, הסוג שאתה מכיר מהבגרות
- **חפיסות מובנות** — אוצר מילים של בגרות (1174 מילים), משפטי מפמ"ר, וסיפור "הנמרים ביום העשירי"
- **עריכת חפיסות** — צור חפיסות חדשות, ייבא מ-CSV/Tab, ייצא ל-Word
- **מילים שלא ידעתי** — סינון אוטומטי של מילים שסומנו כ"לא יודע"
- **חיפוש** — חיפוש חופשי בערבית או עברית בתוך חפיסה
- **קיצורי מקלדת** — רווח/Enter להיפוך, 1/2 לסימון, חיצים לניווט, Escape לסגירה, `/` לחיפוש
- **גסטורות מגע** — החלקה ימינה "יודע", החלקה שמאלה "לא יודע", הקשה להיפוך
- **ייצוא ל-Word** — הורד דף מסודר לכל חפיסה (כולל הגרסה המסוננת בחיפוש)
- **שני כיוונים** — ערבית → עברית או להפך
- **שני מצבי סדר** — לפי הקובץ (ברירת מחדל) או אקראי

## 🚀 הרצה מקומית

פתח את `index.html` בדפדפן. זהו.
(הערה: אם רוצים שה-Service Worker יעבוד כמו שצריך, צריך להגיש דרך שרת HTTP — למשל `python3 -m http.server 8000`.)

## 🌐 פריסה ל-GitHub Pages

1. צור repository חדש ב-GitHub (למשל `quizlets-bagrut`)
2. `git init && git add . && git commit -m "initial" && git branch -M main`
3. `git remote add origin https://github.com/USER/REPO.git && git push -u origin main`
4. ב-GitHub → Settings → Pages → Source: "GitHub Actions"
5. ה-workflow ב-`.github/workflows/pages.yml` ידחוף אוטומטית עם כל push ל-main
6. האתר יעלה ב-`https://USER.github.io/REPO/`

**או** פשוט: Settings → Pages → Source: "Deploy from a branch" → main / root.

## 📁 מבנה

```
.
├── index.html       # ה-SPA עצמה (כולל CSS ו-JS)
├── decks.js         # כל החפיסות המובנות (window.DEFAULT_DECKS)
├── manifest.json    # PWA manifest
├── sw.js            # Service Worker — cache לעבודה אופליין
├── icon-192.png     # אייקון PWA
├── icon-512.png     # אייקון PWA
└── .github/workflows/pages.yml  # אוטומציית פריסה
```

## 🔧 הוספת חפיסות

דרך הממשק: `📚` → "חפיסה חדשה" / "ייבוא".
או ידנית: הוסף אובייקט ל-`decks.js`:

```js
{
  id: 'my_deck',
  name: 'שם החפיסה',
  description: '',
  cards: [
    { ar: 'كِتَاب', he: 'ספר' },
    { ar: 'يَوْم', he: 'יום' },
  ]
}
```

## ⌨️ קיצורי מקלדת

| מקש | פעולה |
|---|---|
| `Space` / `Enter` | היפוך כרטיס |
| `→` | הכרטיס הקודם |
| `←` | הכרטיס הבא |
| `1` | סמן "לא יודע" |
| `2` | סמן "יודע" |
| `1`–`4` | בחירת תשובה באמריקאי |
| `/` | פוקוס על חיפוש |
| `Esc` | סגירת חלון |
